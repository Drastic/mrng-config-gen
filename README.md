# mrng-config-gen
Web-based mRemoteNG connections file generator

## Development
Clone this repo using a git client:
```
git clone https://github.com/Drastic/mrng-config-gen.git
```
Install depencies:
```
node install
```

## Run
Run app:
```
node index.js
```
Open in browser:
```
http://localhost:8000
```
